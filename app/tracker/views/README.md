# Tracker Views

Presentation and orchestration components for the `/tracker` experience.

## Files

- `TrackerAIView.tsx`: main orchestration view (mobile/desktop layout, save wiring, historical view handling).
- `TrackerPanel.tsx`: preview/edit panel controls and tracker display wrapper.
- `TrackerChatPanel.tsx`: chat panel composition.
- `TrackerStatusPanel.tsx`: status banners for validation/generation errors.
- `index.ts`: compatibility barrel exports.
